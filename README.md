# Millo Band - Splash Page

**🌐 Live Site: [https://milloband.com/](https://milloband.com/)**

A lightweight, static landing page for the Millo Band. This page serves as an interactive gateway to the main website, featuring a centered logo with custom CSS hover animations.

## Features
* **Zero Dependencies:** Built with pure HTML and CSS. No heavy frameworks or JavaScript required.
* **Responsive Design:** The logo perfectly centers on all screen sizes and devices.
* **Retro Hover Effect:** Hovering over the logo triggers an animated CRT terminal scanline effect and turns the logo "Baker Miller Pink".
* **Social Ready:** Fully configured with Favicon, Open Graph (Facebook/LinkedIn), and Twitter Card meta tags for clean link sharing.

## How to Use
1. Clone or download this repository.
2. Ensure `Millo - Logo Hat.svg` is in the same directory as the `index.html` file.
3. Open `index.html` in any web browser to view locally.

## Deployment
Because this is a static HTML site, it can be deployed in seconds by dragging and dropping the folder into any static hosting service like [Netlify](https://www.netlify.com/), [Vercel](https://vercel.com/), or [GitHub Pages](https://pages.github.com/).
